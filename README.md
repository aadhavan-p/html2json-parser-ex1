# html2json-parser-ex1
Program to parse HTML to json data object

## Example

`python main.py index.html`

### Output

```python
{
 "capitals": [
  {
   "capital": "Lansing",
   "state": "Michigan"
  },
  {
   "capital": "Sacramento",
   "state": "California"
  }
 ],
 "summary": {
  "numberOfCapitals": 2
 }
}
```
